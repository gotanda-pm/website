Setup
==========================================================================
```sh
git submodule init
git submodule update
npm install -g grunt-cli
npm install
bundle install
carton install
```

Easy usage
==========================================================================
build(CSS/JS)
--------------------------------------------------------------------------
```sh
grunt build
```

publish(Riji)
--------------------------------------------------------------------------
```sh
grunt publish
git commit -av
```

edit and preview
--------------------------------------------------------------------------
```sh
grunt server
```

deploy
--------------------------------------------------------------------------
```sh
grunt deploy
```
